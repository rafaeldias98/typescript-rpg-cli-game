# TypeScript RPG CLI Game (WIP)
A RPG CLI game

## :ballot_box_with_check: Requirements
* [Node.js](https://nodejs.org/) installed, version 10.13 or later

## :scroll: Install dependencies
```sh
$ npm install
```

## :construction: Start Development Server
```sh
$ npm run dev
```
